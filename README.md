📋 Gestion de Tâches

Un projet simple de gestion de tâches (To-Do List) développé avec *HTML*, *CSS* et *JavaScript*.  
Créé par *Ir Bwaha* dans le cadre de la constitution d’un portfolio en génie informatique.

🔍 Fonctionnalités

- Ajouter une tâche
- Supprimer une tâche
- Interface simple, rapide et responsive

🧰 Technologies utilisées

- HTML5
- CSS3
- JavaScript (vanilla)

🚀 Comment exécuter le projet

1. Clone le dépôt :

```bash
git clone https://github.com/IrBwaha/gestion-de-taches.git
```

2. Ouvre le fichier `index.html` dans ton navigateur.

Aucune dépendance requise.

🎯 Objectif du projet

Ce micro-projet a pour but de démontrer les compétences de base en développement front-end, la gestion DOM et l’interactivité utilisateur.
